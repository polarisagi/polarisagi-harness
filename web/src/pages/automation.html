触发器
通过 Webhook 或外部事件触发 Agent 执行——在自动化任务中将触发类型设为"Webhook"后，在此查看并管理入站端点
Webhook 入站端点
在"设置 → 渠道"中创建 Webhook 渠道，然后在自动化任务中将触发类型选为 Webhook 并绑定对应渠道。
入站地址格式：POST /v1/webhooks/webhook/{channelID}，需附带 HMAC-SHA256 签名头 X-Hub-Signature-256。
